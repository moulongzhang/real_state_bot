## ① 青色申告の届出手順と期限

### 1-1. 青色申告とは

不動産所得がある個人が「青色申告」を選択すると、最大 **65万円の特別控除**（複式簿記＋e-Tax利用時）を受けられるほか、**赤字の3年間繰越控除**や**専従者給与の経費算入**などの大きなメリットがあります。

| 控除額 | 条件 |
|--------|------|
| **65万円** | 複式簿記 ＋ e-Tax（電子申告）または電子帳簿保存 |
| **55万円** | 複式簿記 ＋ 書面提出 |
| **10万円** | 単式簿記（簡易簿記） |

### 1-2. 届出に必要な書類

| 書類名 | 入手先 |
|--------|--------|
| **所得税の青色申告承認申請書** | 国税庁HP（PDF）／最寄り税務署／e-Tax |
| 開業届（個人事業の開業届出書） | 未提出の場合は同時提出 |

### 1-3. 申請書の記入項目

```
記入項目:
 ├─ 納税地（住所地）
 ├─ 氏名・生年月日・職業
 ├─ 所得の種類 → 「不動産所得」にチェック
 ├─ 事業開始日（賃貸開始日）
 ├─ 簿記方式 → 「複式簿記」を推奨
 └─ 備付帳簿名 → 現金出納帳・預金出納帳・総勘定元帳・仕訳帳 等
```

### 1-4. 提出期限（★最重要）

| ケース | 提出期限 |
|--------|----------|
| **通常** | 青色申告をしたい年の **3月15日** まで |
| **年途中（1/16以降）に新規開業** | 開業日から **2ヶ月以内** |
| **相続で事業承継** | 相続開始日に応じた期限あり |

> ⚠️ **期限を1日でも過ぎると、その年は青色申告不可**。翌年からの適用になります。

### 1-5. 提出方法

1. **e-Tax**（電子申告）— マイナンバーカード＋PC/スマホで完結
2. **税務署へ持参** — 控えにも受付印をもらう
3. **郵送** — 返信用封筒を同封し、控えの返送を依頼

### 1-6. 手続きフロー

```
Step 1: 国税庁HPまたは税務署で「青色申告承認申請書」を取得
    ↓
Step 2: 必要事項を記入（上記参照）
    ↓
Step 3: 期限内に税務署へ提出（e-Tax/持参/郵送）
    ↓
Step 4: 会計ソフト（freee・弥生・マネーフォワード等）で複式簿記の記帳開始
    ↓
Step 5: 翌年2/16〜3/15に青色申告決算書＋確定申告書を提出
```

## ② 経費として認められる項目一覧

### 2-1. 不動産所得の計算式

```
不動産所得 ＝ 総収入金額 − 必要経費
```

**収入に含めるもの**: 家賃、共益費、礼金、更新料、返還不要の敷金・保証金、駐車場代 等

### 2-2. 経費項目一覧（○ 認められる / × 認められない）

#### ○ 認められる経費

| カテゴリ | 具体的な項目 | 備考 |
|----------|-------------|------|
| **税金** | 固定資産税、都市計画税、不動産取得税、登録免許税、印紙税 | 所得税・住民税は ×（対象外） |
| **管理費** | マンション管理費、管理組合費、管理会社委託料 | 全額経費OK |
| **修繕費** | 原状回復工事、壁紙張替え、設備修理、ハウスクリーニング | 支払った年に一括経費計上 |
| **減価償却費** | 建物本体、付帯設備（エアコン・給湯器等）の償却 | 法定耐用年数で按分（後述） |
| **借入金利息** | ローンの利息部分 | **元本返済分は経費 ×** |
| **保険料** | 火災保険、地震保険、施設賠償責任保険 | 長期一括払いは年数按分 |
| **交通費** | 物件視察、管理会社訪問、セミナー参加の電車代・ガソリン代 | 業務関連性が必要 |
| **通信費** | 携帯電話・インターネット（業務利用分） | 家事按分が必要 |
| **新聞図書費** | 不動産投資関連書籍、新聞、セミナー受講料 | 業務に直接関連するもの |
| **広告宣伝費** | 入居者募集の広告費、仲介手数料 | 全額経費OK |
| **専門家報酬** | 税理士報酬、司法書士報酬、弁護士費用 | 不動産業務に関連するもの |
| **消耗品費** | パソコン（10万円未満）、文具、プリンター用紙 | 業務使用分 |
| **接待交際費** | 管理会社担当者との打合せ飲食 | 過度なものは否認リスクあり |
| **車両費** | 自動車税、ガソリン代、車検代、駐車場代 | 家事按分が必須 |
| **その他** | 立退料、諸手数料、振込手数料 | 業務関連に限る |

#### × 認められない経費

| 項目 | 理由 |
|------|------|
| 所得税・住民税 | 税金の中でも対象外 |
| ローン元本返済分 | 経費ではなく「負債の返済」 |
| 私的な支出（家族旅行・趣味等） | 業務と無関係 |
| 罰金・違反金 | 法律上の罰則 |
| スーツ代（原則） | 不動産業務との関連性が弱い |

### 2-3. 減価償却の基本

```
年間減価償却費 ＝ 建物取得価額 × 償却率

【法定耐用年数の例】
  鉄筋コンクリート（RC）住宅: 47年 → 償却率 0.022
  重量鉄骨造          住宅: 34年 → 償却率 0.030
  木造              住宅: 22年 → 償却率 0.046

【中古物件の簡便法】
  残存耐用年数 ＝ (法定耐用年数 − 経過年数) ＋ 経過年数 × 20%
  ※ 法定耐用年数を全て経過 → 法定耐用年数 × 20%

（例）築25年の木造アパート:
  耐用年数 22年 全経過 → 22 × 20% ＝ 4年（端数切捨て）
  建物取得価額 1,000万円 → 年間 約250万円 の減価償却費
```

> **注意**: 土地は減価償却できません。取得価額を「建物」と「土地」に適正に按分する必要があります。

### 2-4. 家事按分の考え方

プライベートと業務で共用するものは「合理的な基準」で按分します。

| 対象 | 按分基準の例 |
|------|-------------|
| 携帯電話 | 通話履歴から業務利用の割合（例: 30%） |
| 自動車 | 走行距離の業務割合（走行記録をつける） |
| 自宅事務スペース | 面積按分（例: 全体の10%が事務スペース） |

## ③ 確定申告書の記入例

### 3-1. 必要な書類セット

| 書類 | 用途 |
|------|------|
| **確定申告書（第一表・第二表）** | 所得・控除・税額の計算 |
| **収支内訳書（不動産所得用）**（白色の場合） | 収入と経費の内訳 |
| **青色申告決算書（不動産所得用）**（青色の場合） | 複式簿記ベースの損益・貸借対照表 |
| 源泉徴収票 | 給与所得がある場合 |
| 各種控除証明書 | 社会保険料・生命保険料・地震保険料等 |

### 3-2. 記入の流れ（青色申告の場合）

  │                                          │
  │ ② 必要経費                              │
  │    管理費:       360,000円               │
  │    修繕費:       150,000円               │

  │    保険料:        80,000円               │
  │    交通費:        60,000円               │

  │    ─────────────────                     │

  └──────────────────────────────────────────┘

【ステップ2】確定申告書 第一表への転記

    不動産所得:  1,710,000円（決算書⑤から転記）
    ─────────────────
    合計所得:    6,710,000円

  所得控除:
    基礎控除:     480,000円
    社会保険料:   700,000円

    ─────────────────
    控除合計:   1,220,000円

  課税所得金額:  5,490,000円

### 3-3. 損益通算の活用

不動産所得が赤字の場合、**給与所得など他の所得と相殺（損益通算）**できます。

```
（例）不動産所得 −500,000円 ＋ 給与所得 5,000,000円
   → 合計所得 4,500,000円 に圧縮 → 所得税・住民税が減少
```

> ⚠️ 土地取得に係るローン利息部分は損益通算の対象外（制限あり）

## ④ e-Taxでの申告方法

### 4-1. 事前準備

| 準備項目 | 詳細 |
|----------|------|
| **マイナンバーカード** | 電子証明書が搭載されたもの（交付時に暗証番号を設定済み） |
| **ICカードリーダー** または **スマホ** | マイナンバーカード読取用（対応スマホならリーダー不要） |
| **利用者識別番号** | 初回は e-Tax サイトで取得（または ID・パスワード方式も可） |

### 4-2. 申告手順（確定申告書等作成コーナー利用）

```
Step 1: 国税庁「確定申告書等作成コーナー」にアクセス
        URL: https://www.keisan.nta.go.jp/

Step 2: 「作成開始」→「e-Taxで提出」を選択

Step 3: マイナンバーカード方式 or ID・パスワード方式を選択
        → マイナンバーカードを読み取り（スマホ or ICカードリーダー）

Step 4: 「所得税」を選択 → 申告書の作成画面へ

Step 5: 各項目を入力
        ├─ 給与所得 → 源泉徴収票の数字を入力
        ├─ 不動産所得 → 「収支内訳書」or「青色申告決算書」画面で
        │               収入・経費を順番に入力
        │               （物件情報・賃貸料・経費の各項目）
        ├─ 所得控除 → 社会保険・生命保険・医療費等を入力
        └─ 税額控除 → 住宅ローン控除等があれば入力

Step 6: 入力内容を確認 → 「送信」ボタンで提出

Step 7: 受付番号・送信結果を確認 → PDF保存・印刷して保管
```

### 4-3. ID・パスワード方式

マイナンバーカードがない場合の暫定措置です。

```
① 最寄りの税務署で本人確認を受ける
② 「ID（利用者識別番号）」と「パスワード」を発行してもらう
③ 確定申告書等作成コーナーで ID・パスワードを入力して申告
```

> **注意**: この方式は将来廃止予定。早めにマイナンバーカードへの切替えを推奨。

### 4-4. e-Tax のメリット

- 青色申告特別控除が **65万円**（書面だと55万円）
- 24時間いつでも提出可能（メンテナンス時間除く）
- 還付金の処理が早い（通常3週間→e-Taxは約2〜3週間）
- 添付書類の提出省略（5年間保管義務あり）

## ⑤ 税理士に依頼する場合の費用相場

### 5-1. 個人の確定申告（不動産所得）

| 依頼内容 | 費用相場（税込目安） |
|----------|---------------------|
| **確定申告のみ（丸投げ）** | 5万〜15万円/年 |
| **記帳代行＋確定申告** | 8万〜20万円/年 |
| **顧問契約（月額）** | 月1万〜1.5万円（年12万〜18万円） |
| **顧問契約＋確定申告** | 年15万〜30万円 |
| **スポット相談** | 30分 5,000〜10,000円 |

### 5-2. 法人の場合

| 依頼内容 | 費用相場（税込目安） |
|----------|---------------------|
| **決算・申告** | 18万〜25万円/年 |
| **顧問料（月額）** | 月2万〜3万円 |

### 5-3. 費用に影響する要因

```
費用が上がる要因:
  ├─ 物件数が多い（3戸以上、10戸以上で段階的に上昇）
  ├─ 法人と個人の両方ある
  ├─ 売買（譲渡所得）の申告がある
  ├─ 消費税申告が必要
  ├─ 記帳を全て丸投げ
  └─ 年度途中の相談が多い

費用を抑える方法:
  ├─ 自分で記帳（会計ソフト利用）→ 申告のみ依頼
  ├─ 資料を整理してから渡す
  ├─ 繁忙期（2〜3月）を避けて早めに依頼
  └─ 複数の税理士から見積もりを取る
```

### 5-4. 税理士を選ぶポイント

- **不動産投資・賃貸業に強い税理士**を選ぶ（専門性が重要）
- 料金体系が明確か（追加料金の有無）
- レスポンスの速さ・コミュニケーションの質
- 節税提案を積極的にしてくれるか
- 税理士報酬は**全額経費計上可能**

## ⑥ よくある間違い・税務調査のポイント

### 6-1. 確定申告でよくある間違い TOP10

| No. | よくある間違い | 正しい処理 |
|-----|---------------|-----------|
| 1 | **収入の計上漏れ**（礼金・更新料・共益費を含めていない） | 家賃以外の収入も全て「総収入金額」に計上 |
| 2 | **修繕費 vs 資本的支出の混同** | 原状回復＝修繕費（一括経費）、価値向上＝資本的支出（減価償却） |
| 3 | **減価償却の計算ミス** | 耐用年数・償却率を正確に。中古物件の簡便法にも注意 |
| 4 | **建物と土地の按分が不適正** | 固定資産税評価額等の合理的基準で按分 |
| 5 | **ローン元本を経費計上** | 経費になるのは利息部分のみ |
| 6 | **私的支出の経費計上** | 家族旅行・私用の食事代等は ×。按分不明確なものも否認される |
| 7 | **家事按分の根拠なし** | 按分比率の根拠資料を残す（走行記録・利用ログ等） |
| 8 | **敷金（返還予定分）を収入計上** | 返還予定の敷金は預り金で、収入ではない |
| 9 | **未収家賃の未計上** | 発生主義：入金がなくても発生した時点で収入計上 |
| 10 | **青色申告承認申請の期限切れ** | 翌年の青色申告が使えなくなる |

### 6-2. 税務調査で重点的にチェックされるポイント

```
【税務調査官が見る主要チェックポイント】

1. 収入の網羅性
   └→ 銀行口座・管理会社の入金記録と申告額が一致するか

2. 経費の妥当性
   ├→ 領収書の「但し書き」と金額
   ├→ 業務関連性の説明ができるか
   └→ 家事按分の合理性

3. 減価償却の正確性
   ├→ 耐用年数の適用が正しいか
   ├→ 取得価額（特に建物と土地の配分）
   └→ 資本的支出 vs 修繕費の判断

4. 名義と実態の一致
   └→ 家族名義で実質本人が運用していないか

5. 大きな金額変動
   └→ 前年と大幅に異なる収入・経費は調査対象になりやすい
```

### 6-3. 税務調査が入りやすいケース

- **無申告**（家賃収入があるのに申告していない）
- **経費率が異常に高い**（同業種・同規模の平均値と大きく乖離）
- **毎年赤字申告を続けている**
- **売買（譲渡）があった年**
- **収入の大きな変動がある**

### 6-4. ペナルティ一覧

| 種類 | 税率 |
|------|------|
| 過少申告加算税 | 10〜15% |
| 無申告加算税 | 15〜20% |
| 重加算税（隠蔽・仮装） | 35〜40% |
| 延滞税 | 年2.4%〜（期間による） |

## ⑦ 節税に効く領収書の整理法

### 7-1. 保管の基本ルール

| 書類 | 保管期間 |
|------|----------|
| 領収書・レシート | **7年間**（青色申告） |
| 請求書・納品書 | 7年間 |
| 契約書・覚書 | 7年間（可能なら永久保管推奨） |
| 帳簿類 | 7年間 |
| 銀行通帳・ローン明細 | 7年間 |

> 白色申告の場合、一部書類は5年間ですが、**統一して7年保管が安全**です。

### 7-2. おすすめ整理術：月別×科目別ファイリング

```
領収書整理の手順:

Step 1: 支払い時にすぐ分類封筒に入れる
        ├─ 「管理費・委託料」
        ├─ 「修繕費」
        ├─ 「交通費」
        ├─ 「書籍・セミナー」
        ├─ 「保険料」
        ├─ 「税金（固定資産税等）」
        └─ 「その他経費」

Step 2: 月末にA4用紙へ貼付
        → 日付順に並べて、科目別に分類して貼る
        → 各用紙に「○年○月分/科目名/合計額」を記入

Step 3: 12ポケットファイルに月別で収納
        → 1月〜12月の12ポケット＋年間まとめ用1ポケット

Step 4: 会計ソフトに入力（または写真撮影してクラウドへ）
```

### 7-3. デジタル管理（電子帳簿保存法対応）

```
【クラウド会計ソフト活用フロー】

  領収書を受領
      ↓
  スマホで撮影（freee/弥生/マネーフォワード等のアプリ）
      ↓
  AI-OCR で自動読取 → 仕訳候補が自動生成
      ↓
  科目・金額を確認して承認
      ↓
  電子データとして保存（電子帳簿保存法要件を満たす形式）
      ↓
  決算書・申告書も自動生成 → e-Tax連携で提出
```

**電子帳簿保存法の注意点**:
- タイムスタンプの付与、または訂正・削除の履歴が残るシステムが必要
- 検索機能（日付・金額・取引先）の確保
- 2024年1月以降、電子取引データは電子保存が原則義務化

### 7-4. 領収書がない場合の対処法

| ケース | 対処法 |
|--------|--------|
| 自販機・電車賃 | **出金伝票**を作成（日付・金額・用途・支払先を記入） |
| レシートを紛失 | クレジットカード明細＋メモで代用（税務署により判断分かれる） |
| ネット決済 | メール領収書・決済履歴をPDF保存 |
| 祝儀・香典 | 出金伝票＋案内状等で記録 |

### 7-5. 節税効果を最大化するチェックリスト

```
□ 全ての経費の領収書を漏れなく保管しているか
□ 「但し書き」に具体的内容が記載されているか
  （× 「お品代」 → ○ 「不動産管理関連書籍代」）
□ 家事按分の根拠資料（走行記録・利用ログ等）を残しているか
□ 修繕費と資本的支出を正しく分類しているか
□ 減価償却資産台帳を更新しているか
□ クレジットカードは事業用と私用を分離しているか
□ 銀行口座も事業用と私用を分離しているか
□ 年末に経費の計上漏れがないか最終チェックしたか
□ 電子帳簿保存法に対応した保存方法になっているか
□ 7年分の書類が安全に保管されているか
```

## 付録: 年間スケジュール

| 時期 | やるべきこと |
|------|-------------|
| **1月** | 前年の帳簿を締める・年間収支を集計 |
| **2月上旬** | 確定申告書類の作成開始、不足資料の確認 |
| **2/16〜3/15** | **確定申告期間**（e-Taxで早めに提出推奨） |
| **3/15** | 青色申告承認申請の期限（翌年分）、振替納税の届出 |
| **4月** | 固定資産税の納付書届く（年4回分割 or 一括） |
| **6月** | 住民税の通知届く（不動産所得分が上乗せ） |
| **毎月** | 領収書の整理・会計ソフトへの記帳 |
| **12月** | 年末の経費チェック・翌年の節税対策検討 |
