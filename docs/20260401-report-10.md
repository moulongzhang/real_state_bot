# 不動産投資調査レポート: 投資対象エリア ハザードマップ災害リスク調査

**作成日**: 2026年04月01日
**レポートID**: 20260401-010
**レポートタイプ**: エリア深堀り（災害リスク評価）

---

## 1. エグゼクティブサマリー

> **結論: 6エリアの災害リスクには明確な優劣がある。投資判断への影響は大きい。**
>
> ハザードマップポータルサイト（https://disaportal.gsi.go.jp/）および各自治体公表のハザードマップを横断調査した結果、**足立区・葛飾区は洪水リスクが「極めて高い」**（荒川・江戸川等の複数河川氾濫で区のほぼ全域が浸水想定区域）。**北区・板橋区は荒川沿い低地部に洪水リスクが集中するが、台地上は比較的安全**。板橋区は崖線沿いに**土砂災害警戒区域が152箇所**と突出して多い。**横浜市青葉区は洪水リスクが限定的で最もバランスが良い**が、丘陵地のため崖地リスクに注意。**川崎市は2019年台風19号で実被害あり**、多摩川沿いの物件は要注意。
>
> **投資適性（災害リスク観点）: 横浜市青葉区 > 板橋区（台地上）≧ 北区（台地上）> 川崎市（内陸部）> 足立区 > 葛飾区**

## 3. エリア別 災害リスク評価

### 3.1 足立区（荒川氾濫リスク）

| リスク項目 | 評価 | 詳細 |
|-----------|------|------|
| **洪水（外水氾濫）** | 🔴 極めて高い | 荒川・利根川・江戸川・中川・綾瀬川・芝川の**6河川**が氾濫リスク対象。想定最大規模（荒川: 3日間632mm）で**区のほぼ全域が浸水**。最大浸水深**5m以上**（2階水没）。浸水継続時間**最大2週間以上** |
| **洪水（内水氾濫）** | 🟡 中程度 | 隅田川・新河岸川流域、中川・綾瀬川圏域の浸水予想区域図あり。ゲリラ豪雨時の下水氾濫リスク |
| **高潮** | 🟡 中程度 | 高潮浸水想定区域に指定（令和6年12月 東京都公表の新想定区域図に基づく） |
| **土砂災害** | 🟢 なし | 区内に**土砂災害警戒区域の指定なし**（公式確認済） |
| **津波** | 🟢 なし | 津波災害警戒区域の指定なし |
| **液状化** | 🔴 高い | 荒川低地の沖積層地盤。東京都液状化予測図で大部分が「液状化の可能性が高い」地域 |

**ハザードマップ出典**: 足立区洪水・内水・高潮ハザードマップ（令和7年4月改訂）
- URL: https://www.city.adachi.tokyo.jp/kikaku/bosai/bosai/hazard-map-k.html
- 荒川洪水浸水想定区域図: 国土交通省（平成28年5月公表）
- 高潮浸水想定区域図: 東京都（令和6年12月公表）

**投資への影響**:
- 1階店舗・事務所は浸水リスクで賃料下落・空室リスク大
- 火災保険の水災特約は必須（保険料上昇要因）
- 江東5区大規模水害広域避難対象エリア（最大250万人避難想定）
- **重要事項説明で水防法に基づく説明義務あり**

- **台地上（赤羽台・王子駅周辺の高台）の物件は災害リスクが大幅に低い** → 投資適性高い

- 崖線沿いは土砂災害リスクのため現地確認必須
- 立地の「高低差」が投資判断の決定的要因

| リスク項目 | 評価 | 詳細 |

- 洪水: https://www.city.itabashi.tokyo.jp/bousai/bousai/map/1005633.html
- 土砂: https://www.city.itabashi.tokyo.jp/bousai/bousai/map/1005634.html
- 浸水想定区域図の変更はなし（最新版でも内容同一）

**投資への影響**:
- **土砂災害警戒区域 152箇所は6エリア中最多** → 崖線沿い物件は避けるべき
- 台地上で崖線から離れた物件は洪水・土砂の両リスクが低い
- 特別警戒区域（レッドゾーン）内は建築規制・開発許可制のため資産価値への影響大

### 3.5 横浜市青葉区（鶴見川水系）

| リスク項目 | 評価 | 詳細 |
|-----------|------|------|
| **洪水（外水氾濫）** | 🟡 中程度（河川沿い限定） | 鶴見川水系（鶴見川本川・恩田川・奈良川等）の氾濫リスク。ただし青葉区は**上流域で標高が高く**、浸水想定区域は河川沿いの谷底低地に限定。大部分の住宅地は浸水想定区域外 |
| **洪水（内水氾濫）** | 🟢 低い | 丘陵地のため排水勾配が確保され、内水氾濫リスクは低い |
| **高潮** | 🟢 なし | 内陸丘陵地のため高潮影響なし |
| **土砂災害** | 🟠 あり（丘陵地特有） | 丘陵地帯のため急傾斜地が散在。すすき野・美しが丘・奈良等の崖線沿いに土砂災害警戒区域あり。ただし板橋区ほど密集していない |
| **津波** | 🟢 なし | 内陸のため対象外 |
| **液状化** | 🟢 低い | 丘陵台地が主体のため全般的に低リスク。谷戸・谷底低地部は中程度 |

**ハザードマップ出典**: 横浜市洪水ハザードマップ・横浜市土砂災害ハザードマップ / 国土交通省ハザードマップポータル
- ハザードマップポータル: https://disaportal.gsi.go.jp/maps/
- 鶴見川は国の「総合治水対策特定河川」指定（遊水池・調節池整備済）

**投資への影響**:
- **6エリア中、災害リスクが最もバランス良く低い**
- 鶴見川水系は総合治水対策が進んでおり、遊水池（鶴見川多目的遊水地等）による治水効果大
- 崖地・急傾斜地の物件は個別にハザードマップ確認必須
- 丘陵台地上の物件は洪水・液状化とも低リスク → **投資適性最高**
- 前回レポート（20260401-009）の対象物件「すみよし台RC一棟」は台地上に位置し災害リスク低い

- 内陸高台の物件であれば投資適性は確保される

## 5. 投資判断・推奨アクション

### 災害リスク観点の投資適性ランキング

1. **横浜市青葉区（丘陵台地上）** — 洪水・液状化とも低リスク。鶴見川治水対策済。**最も安全**
2. **北区（赤羽台・王子台地上）** — 台地上限定で洪水・液状化リスク回避可能。崖線注意
3. **板橋区（台地上・崖線外）** — 台地上は洪水低リスクだが土砂災害警戒区域152箇所に要注意
4. **川崎市（宮前区・麻生区等の丘陵部）** — 内陸高台なら低リスクだが2019年実被害の心理的影響残る
5. **足立区** — 区のほぼ全域が複数河川の浸水想定区域。回避困難
6. **葛飾区** — 3河川に挟まれた構造的脆弱性。6エリア中最高リスク

### ネクストアクション

- [ ] 個別物件検討時に**重ねるハザードマップ**（https://disaportal.gsi.go.jp/maps/）で番地単位の浸水深・土砂災害区域を確認
- [ ] 足立区・葛飾区の物件は**火災保険の水災特約保険料を事前見積**し、CF計算に反映
- [ ] 北区・板橋区は**標高・海抜を確認**（台地上 vs 低地で災害リスクが全く異なる）
- [ ] 板橋区物件は**土砂災害(特別)警戒区域内かどうか**を必ず確認（特別警戒区域は建築規制あり）
- [ ] 川崎市物件は**2019年台風19号の浸水実績区域**との照合を実施
- [ ] 全物件で**液状化予測図**（東京都・横浜市・川崎市公表）を個別確認

## 6. リスク・注意事項

| リスク | 影響度 | 対策 |
|--------|--------|------|
| 荒川決壊（足立区・葛飾区・北区・板橋区） | 極大 | 高層階投資・台地上物件選定・水災保険必須 |
| 土砂災害（板橋区崖線沿い） | 大 | 土砂災害警戒区域の回避。特別警戒区域は投資不適 |
| 液状化（足立区・葛飾区・川崎臨海部） | 大 | 地盤調査・杭基礎確認。RC造でも基礎沈下リスク |
| 多摩川氾濫再発（川崎市） | 大 | 2019年実績エリアの回避。ハザードマップと浸水実績の二重確認 |
| 気候変動による降雨量増加 | 中長期 | 想定最大規模を上回る降雨の可能性。保守的判断推奨 |
| 重要事項説明義務（水防法） | 制度リスク | 浸水想定区域内の物件は宅建業法で説明義務あり。売却時の心理的影響 |
