### 第1位: オリックス銀行 ★★★★★

**融資期間の長さが最大の武器。1棟目に最も適した金融機関**

| 項目 | 内容 |
|------|------|
| 金利 | 変動 2.4〜3.0%（団信込み） |

| 融資可能額 | 11,200万〜14,000万円（LTV80〜100%） |
| 自己資金 | 0〜2,800万円（フルローン交渉可能） |
| 年収要件 | 500万円以上（実質700万〜推奨） |
| 手数料 | 融資額×2.2%（約246〜308万円） |

**メリット:**
- 🔥 **融資期間が最大の武器**。RC残存28年を超え30年も交渉可能
- 八王子は首都圏対象エリア内 → 問題なく取扱い可能
- 1〜2棟目の初心者サラリーマン投資家に最も使いやすい
- 審査スピードが速く物件を押さえやすい
- 法人（資産管理会社）でも個人と同条件
- 口座開設不要（既存銀行口座で返済可）
- 保証料不要
- 年収1,600万・借入金0・資産3,000万は高属性 → フルローン交渉の余地あり

**デメリット:**
- 変動金利のみ（固定なし）→ 金利上昇リスク
- 金利は地銀より高め（2.4〜3.0%）
- 融資後に物件価格×20%の金融資産を手元に残す必要あり（2,800万 → 保有3,000万でギリギリ）

## 💡 融資交渉のポイント

### この物件の「融資に有利な材料」

1. **土地値比率84.6%** → 積算評価が非常に出やすい。路線価ベースの担保価値が高い
2. **RC残存28年** → 融資期間に十分な余裕（25〜30年が交渉可能）
3. **八王子駅徒歩5分・2路線利用可** → エリア力あり（JR八王子駅は1日約85,000人利用）
4. **近隣商業地域・容積率300%** → 建て替え余地があり資産性が高い
5. **所有権・公道接道** → 減額要因なし
6. **築19年で比較的新しい** → 大規模修繕の猶予あり、設備面の不安が少ない

### この物件の「融資で不利な材料」

1. **八王子市は23区外** → 一部銀行で評価減（23区・横浜に比べ格落ち扱い）
2. **利回り5.82%** → 高利回り重視の銀行（きらぼし7.5%〜）では弱い
3. **8戸は大規模物件ではない** → スケールメリットが限定的
4. **現況利回り5.22%** → 現況ベースの審査では返済比率が厳しくなる
