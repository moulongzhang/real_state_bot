- 保証料不要

- 変動金利のみ（固定なし）→ 金利上昇リスク
- 1億超は本部審査でやや時間がかかる
- 金利は地銀より高め

## 💡 融資交渉のポイント

### この物件の「融資に有利な材料」
1. **積算評価93.8%** → 積算割れがほぼなく、銀行評価が出やすい
2. **土地比率67.5%** → 積算の7割が土地で下値リスク限定的
3. **RC残存22年** → 融資期間20年以上が引ける
4. **ハザードリスク低** → 標高54m丘陵地、液状化・洪水リスク低
5. **所有権・整形地** → 減額要因なし
6. **路線価上昇傾向** → +0.5%/年で積算が微増見込み

### この物件の「融資で不利な材料」
1. **恩田駅の乗降客数974人/日** → エリア力の弱さは銀行も認識
2. **1R×11戸** → 単身者の回転率の高さをどう説明するか
3. **返済比率が高め** → 頭金を積み増すか融資期間の延長が必要
4. **利回り6.9%** → 静岡銀行のB②基準7.0%にわずかに未達

### 指値交渉との連動
- **1億1,000万円**に指値 → 利回り7.15%に改善 → 静岡銀行B②基準クリア
- **1億500万円**に指値 → 利回り7.5%に改善 → きらぼし銀行の好条件ゾーン
- **1億円**に指値 → 利回り7.87% → 全行で最も有利な条件を引き出せる
