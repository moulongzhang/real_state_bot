## 2. 建物積算

### 残存価格率

```
残存価格率 = 残存耐用年数 ÷ 法定耐用年数
           = 22年 ÷ 47年
           = 0.4681（46.81%）
```

## 5. 積算評価のポイント

### ✅ ポジティブ

| 項目 | 評価 |
|------|------|
| **積算価格率93.8%（6.2%の積算割れ）** | 首都圏RC一棟では一般的な水準。大幅な積算割れではない |
| **土地比率67.5%** | 積算の約7割が土地 → 下値リスクが限定的 |
| **路線価上昇傾向** | +0.5%/年 → 積算価格は今後も微増の見込み |
| **所有権・整形地** | 借地権や不整形地のような減額要因なし |

### ⚠️ 注意点

| 項目 | 内容 |
|------|------|
| **奥行補正率未確定** | 実際の間口・奥行が不明。現地確認で0.95〜1.00の範囲 |
| **建物の再調達原価** | 銀行ごとに18〜34万円/㎡と大きな幅。事前打診で確認要 |
| **経年劣化の実態** | 計算上は46.8%残存だが、修繕状況により銀行評価が上下 |
| **低層住専の容積率80%** | 建て替え時に現在の建物規模を再現できない可能性あり |

## 6. 結論

```
┌─────────────────────────────────────────────┐
│                                             │
│   積算価格:  約1億694万円（標準シナリオ）       │
│   売出価格:  1億1,400万円                     │
│   積算価格率: 93.8%                           │
│                                             │
│   → 売出価格は積算価格をやや上回る水準         │
│   → 約706万円（6.2%）の乖離は許容範囲内       │
│   → 指値交渉の材料として「1億〜1億800万」が     │
│     積算評価上の妥当レンジ                    │
│                                             │
└─────────────────────────────────────────────┘
```
